# cas-fee-projekt1
Eine Applikation zum Verwalten von Notizen im Rahmen des CAS FEE 2017.

## Installation

```bash
git clone https://github.com/alc-hsr/cas-fee-projekt1.git
cd cas-fee-projekt1
npm install
```

## Start

### Web-Server starten

```bash
node server.js
```

### Client starten

http://localhost:3000 aufrufen.
